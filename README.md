# homebrew-tap

Homebrew tap for [AleDeclerk](https://github.com/AleDeclerk)'s tools.

## Available

| Cask | Description | Source |
|------|-------------|--------|
| `t4p` | Free LLM tokens for the rest of us | [tokensforthepeople](https://github.com/AleDeclerk/tokensforthepeople) |

## Install

```bash
brew install AleDeclerk/tap/t4p
```

Or tap explicitly:

```bash
brew tap AleDeclerk/tap
brew install t4p
```
